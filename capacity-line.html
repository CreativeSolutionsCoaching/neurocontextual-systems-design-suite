What These Choices Actually Mean
Structure It — T1
Using external support: asking for help, breaking things into steps, using a system or routine. This is the highest-capacity move because it offloads the work instead of absorbing it.
Push Through — T2
Handling it internally: willpower, staying composed, not letting it show. It works — but it costs capacity every time because you're carrying it alone.
Let It Hit — DX
No move at all — the situation lands without any response. Sometimes that's all that's available. But when it's a pattern, it's usually a sign that capacity was already too low to do anything else.
