How This Works
All tools use NSDS Profile codes — the four-initial composite generated by the NSDS Profile assessment (e.g., W · N · S · C). Employees complete the profile; their code is the only information needed here. No diagnosis. No clinical history. No personal data beyond what they choose to share.
